# [York University Digital Library](https://digital.library.yorku.ca)

## Description:

YUDL is deployed on a single virtual machine, and the Fedora data directory is deployed to a 2T NFS mount.

* OS: Ubuntu 12.04.02 LTS x64
* 4 CPUs
* 24G RAM

## Tomcat webapps:

* Adore-Djatoka
* Fedora
* Fedora GSearch
* Open Wayback
* Solr
